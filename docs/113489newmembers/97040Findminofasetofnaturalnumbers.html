And I answer myself…

lemma smallest_element (S: set ℕ): S.nonempty → ∃ n : ℕ, ∀ m < n, m ∉ S := begin
  intro H,
  use (nat.find H),
  intro m,
  apply nat.find_min H,
end

seems to work just fine.

But I wonder if there is a more canonical way.
