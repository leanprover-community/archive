I resolve Q2. I misread how or.intro is defined. Its first input should be the type of the term NOT being exacted. Fixing this now makes the proof compile successfully. 

example : (p ∨ q) ∨ r ↔ p ∨ (q ∨ r) :=

    (assume h : (p ∨ q) ∨ r, or.elim h
