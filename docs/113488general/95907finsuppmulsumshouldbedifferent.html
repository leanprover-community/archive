Topic: `finsupp.mul_sum` should be different

{% raw %}

Abhimanyu Pallavi Sudhir (Nov 25 2019 at 18:25):

finsupp.mul_sum says:

finsupp.mul_sum :  ∀ {α : Type u_1} {β : Type u_2} {γ : Type u_3} [_inst_1 : semiring β] [_inst_2 : semiring γ] (b : γ) (s : α →₀ β) {f : α → β → γ},
    b * finsupp.sum s f = finsupp.sum s (λ (a : α) (c : β), b * f a (⇑s a))

Isn't this more useful:

finsupp.mul_sum :  ∀ {α : Type u_1} {β : Type u_2} {γ : Type u_3} [_inst_1 : semiring β] [_inst_2 : semiring γ] (b : γ) (s : α →₀ β) {f : α → β → γ},
    b * finsupp.sum s f = finsupp.sum s (λ (a : α) (c : β), b * f a c)
