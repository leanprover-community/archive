Does anyone know if this is already in the library?

/-- A subset of a subtype can be thought of as a subset of the type. -/
def set_of_subtype_set {α : Type*} {P : α → Prop} (x : set {a : α // P a}) : set α :=
{a : α | ∃ (h : P a), (⟨a,h⟩ : {a : α // P a}) ∈ x}
